
# 校园墙系统 - 产品需求文档 (PHP版)

## 1. 产品概述
校园墙系统是一个集信息发布、分享、交流于一体的校园社交平台，采用 PHP 8.0+ 和 MySQL 5.7 技术栈开发，包含前台用户页面和后台管理系统。

## 2. 核心功能

### 2.1 用户角色
- **普通用户**：无需注册登录（可选），浏览、发布动态、点赞、评论
- **管理员**：登录后台，管理内容、用户、话题等

### 2.2 功能模块

#### 前台功能
1. **首页**：轮播公告、话题分类、动态列表、搜索
2. **发布页**：内容编辑、图片上传、话题选择
3. **详情页**：动态详情、点赞、评论列表、发布评论
4. **个人中心**：我的动态、个人设置

#### 后台管理功能
1. **登录**：管理员身份验证
2. **仪表盘**：数据统计、最近动态
3. **动态管理**：查看、审核、删除、编辑动态
4. **评论管理**：查看、删除评论
5. **话题管理**：添加、编辑、删除话题
6. **管理员设置**：密码修改

### 2.3 页面详情

| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 首页 | 轮播图 | 展示校园公告 |
| 首页 | 话题标签 | 按话题筛选动态 |
| 首页 | 动态列表 | 瀑布流展示用户内容 |
| 发布页 | 编辑区 | 文本编辑、图片上传 |
| 详情页 | 内容展示 | 完整内容、图片、互动 |
| 后台登录 | 登录表单 | 管理员登录 |
| 后台仪表盘 | 统计面板 | 数据概览 |
| 后台动态管理 | 管理列表 | 内容审核与管理 |

## 3. 核心流程
- 用户浏览 → 查看详情 → 点赞/评论
- 用户发布 → 选择话题 → 提交内容 → 等待审核（可选）
- 管理员登录 → 管理内容 → 审核通过/删除

## 4. 界面设计
- 前台：青春活力，蓝色主调，移动端优先
- 后台：简洁专业，蓝白配色，响应式设计
